# BLogProject
📌 Small Blog Platform  Technologies: Django, Django ORM, SQLite, Bootstrap  🚀 Features:  🔑 User registration and authentication (Django Auth)  ✍️ Create, edit, and delete posts  💬 Comment on posts  🖼️ Upload media files (images)  📄 Simple and clean frontend (Bootstrap)  ⚙️ Admin panel to manage users and posts
